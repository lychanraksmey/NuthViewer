# What is this?

Turn a pair of phones into a video nanny cam and monitor your child from anywhere!
With this NuthViewer baby monitor app, you can use your old phone as a baby camera and have a piece in mind when you are not around. 
Like, If your baby is asleep and you are in another room or sth.

NuthViewer - a must have baby monitor app for all parents! 


# Features
- [x] Made by a parent for parents, nothing fancy but nothing fishy
- [x] Connection via IP (Manual Connection), via QR scanning, via Wifi router scanning (Auto Connection)
- [x] Hotspot (for Offline usage)
- [x] Motion Detection Alert
- [x] Push to talk (Send voice from Parent device to Child device)
- [x] Listening (Send voice from Child device to Parent device)
